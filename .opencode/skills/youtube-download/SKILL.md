# YouTube Download Best Practices

When using yt-dlp:

- Validate URLs before downloading
- Use safe output filenames
- Limit maximum file size
- Use temp directories for downloads
- Clean temporary files after sending

Prefer options:

- format selection
- audio extraction with ffmpeg
- progress hooks for status updates
